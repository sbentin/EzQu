EzQu originated in 2008 as a fork of JaQu,
a tool developed by Thomas Mueller, the author of H2 Database.

The original code was published under the following copyright
message:

Copyright 2004-2011 H2 Group
H2 License version 1.0
http://h2database.com/html/license.html

After jaQu was abandoned due to a lack of attention,
EzQu forked the project and has since undergone significant development,
following a different path from the original JaQu project.
Today, EzQu is a well-maintained, supported, and extended 
library that has been deployed in large projects using 
MySql, MariaDB, Oracle, PostgresDB, DB2, MS SqlServer, and H2.
